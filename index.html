Codebar Pairing Helper

Drag the following bookmarklet link to your bookmarks:

Pair codebar students

When pairing students at a codebar meet, click the link in your bookmarks to enable some additional pairing functionality.

Features

  • Automatically sort students by attendance and subject
  • Drag and drop students to reorder manually
  • Edit a student's subject text inline
  • Copy list of student attendees to clipboard, to easily migrate to your text-editor if desired
  • Visually indicate students' subjects using background-colour hues
  • Any changes are saved to localStorage, so they'll persist when refreshing
  • Needs to be restarted every time you sign in a student (due to the page refresh)
  • Has only been tested in Chrome on desktop

Github contributions welcome
